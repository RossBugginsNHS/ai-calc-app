## Annotations & Accessibility Notes

### Touch Targets

- **Minimum**: 44x44px (WCAG 2.5.5)
- **Comfortable**: 48x48px (Material Design recommendation)
- **Calculator buttons**: 64-80px (depending on viewport)

### Color Contrast

- **Text on background**: Minimum 4.5:1 (WCAG AA)
- **Large text (18pt+)**: Minimum 3:1 (WCAG AA)
- **Interactive elements**: Ensure sufficient contrast in all states (default, hover, focus, active, disabled)

### Keyboard Navigation

- All interactive elements must be keyboard accessible
- Visible focus indicators (2px outline, offset 2px)
- Tab order follows logical reading order
- Keyboard shortcuts documented and accessible via "?" key

### Screen Reader Support

- Semantic HTML (buttons, headings, landmarks)
- ARIA labels for icon-only buttons
- ARIA live regions for calculation results and error messages
- Skip links ("Skip to calculator", "Skip to history")

### Animation & Motion

- Respect `prefers-reduced-motion` media query
- Disable animations for users who prefer reduced motion
- Transitions: 200-300ms (fast enough, not jarring)
